Växlaren nere till höger (A/B/C) är **gransknings-chrome, inte del av designen**.

## Copy (gemensam, ampy-rost)

- Eyebrow: **"Elfirma i hela Sverige"** (nationellt anspråk per ägardirektiv 2026-07-18)
- H1: **"El i hemmet, gjort ordentligt."** (gradient-highlight = befintligt brand-mönster)
- Sub: **"Egna behöriga elektriker för allt från elfel och elcentraler till laddbox och batterilagring. Fast pris, ärliga besked och hjälp i hela Sverige."** (service → laddbox → batteri = kommersiell prio; "fast pris" = kanon-verbatim)
- Primär CTA: **"Boka kostnadsfri rådgivning"** → /kontakt/ · Sekundär (nedtonad): **"Akut? Ring 010-265 79 79"** → tel:
- Proof: **"5,0 på Google"** (länkad, ampy.se/omdomen/) + **"Över 3 000 installationer per år"** (owner-confirmed 2026-07-10)

## Fakta & öppna punkter

- Inga lånade/påhittade tal. Alla siffror ur `ampy-foretagsdata` + ägardirektiv.
- `[GAP]` Google-betygets aktuella siffra/antal — bekräfta före skarp Bricks-sättning.
- `[GAP]` CTA-destination: /kontakt/ nu; nya huvudformuläret när det skeppas.
- Fotot är 1000×827 (ägarlevererat). För **B (full-bleed)** i produktion behövs en högre upplösning (~2000px+) — A och C klarar sig.
- Verifierat med egna ögon (ampy-syn): desktop 1440 + mobil 390, alla tre riktningar.

## Nästa steg

1. Ägaren väljer riktning (eller beställer hybrid/justeringar).
2. Vald riktning konverteras till Bricks + FluentSnippets (mini-menyn/block 2 ligger direkt under heron på sajten).
